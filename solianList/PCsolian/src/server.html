2. 甲方在搜恋平台上填写的是真实、有效的联系方式（包括但不限于手机、电子邮件、MSN/QQ或邮寄地址）并随时将更新信息通过乙方确保乙方随时能与甲方保持畅通联系，但由于甲方自身原因致使其连续30日（自甲方不对乙方进行反馈之日开始起算）无法和乙方安排的符合甲方择偶标准的婚恋对象见面，视为甲方已接收乙方服务，乙方不承担任何责任。
3. 确保服务期内所有的服务项目均由本人亲自参与并向乙方提供真实、及时的反馈。甲方不得将本协议项下的权利义务转让给任何第三方，否则乙方有权随时单方面终止协议且无须退还牵缘服务费用。 4. 在接受服务及交友中遵守法律法规和社会公德，否则应承担可能应发的法律责任。若给乙方和/或第三方造成损失的，甲方须进行相应赔偿。如甲方有欺诈行为或利用乙方提供的婚姻介绍服务从事违法行为的，乙方有权单方解除本合同，且不予退还收取的保证金费用。
5. 甲方应具备独处判断能力和完全民事行为能力，在征婚交友过程中与婚恋网对象发生的任何纠纷，或遭受的任何人身或财产损失与乙方无关。

二、乙方之权利与义务
1.甲方向乙方支付约会保证金后乙方向甲方提供以下四种服务：
（1）终身享受无限次线上VIP免费自主匹配，每年12次线下红娘约会接力，免费提供优美约会场景
（2）享受免费情感心理疏导1次
（3）免费个人形象打造1次
（4）脱单后免费婚礼策划1次
2.乙方有权验证甲方会员提交的身份证相关信息，由于国内相关信息系统联网查询的局限性及法律对于个人隐私信息的保护制度，乙方了解的其他信息可能无法得到第三方权威机构的佐证，而只能依赖于推荐人员愿意提交的相关证件之复印件或书面声明。甲方应对于此种事实予以理解并认可。在与推荐人员的接触时，甲方应对相关信息进行独立判断，对于乙方了解信息与实际情况的偏差亦应予以合理的预期和理解。
3.乙方在服务过程中，做好跟踪服务记录，并定期与甲方确认服务进程与结果。
4.乙方须对甲方所提供的个人身份信息包括但不限于：会员姓名、联系方式、婚姻状况、学历及工作证明承担保密义务。
5.乙方保证仅将甲方所提供的个人身份信息用于与本协议目的有关的事务范 围内，除此之外，未经甲方许可，不得基于非婚恋交友目的使用甲方个人身份信息。

三、服务的终止
1.甲方有权通过搜恋平台或其它非搜恋平台渠道寻找合适伴侣，甲方在获得国家民政部门颁发的结婚证书之后，向乙方提交申请退还1000元保证金，乙方对甲方的结婚证明材料进行核实，若情况属实，乙方保证在甲方发起申请的7个工作日内退还甲方的1000元保证金。
2.有下列情形之一的，乙方可单方终止本协议项下的相关服务，并有权拒绝退还甲方的保证金：
（1）甲方在申请牵缘服务时在搜恋平台提供的个人资料被发现为不真实的;
（2）恶意发布约会邀请，对搜恋平台其他会员造成人身伤害或财产损失的行为；
（3）甲方单方面要求终止协议的，其保证金不予退还，且乙方在收到甲方要求的次日起终止对甲方的服务。
（4）甲方在本协议服务期内依法被追究刑事责任的；
（5）甲方发生疾病、伤残或意外及其它不可抗力因素造成乙方无法服务的；
（6）本协议的乙方服务仅限中国大陆地区，若因甲方不在服务区域内则乙方有权拒绝服务而不承担其它任何责任。

五、违约责任
1.甲、乙双方应正当行使权利，履行义务，保证本协议的顺利履行。
2.任何一方未依本协议约定履行义务的，应当承担违约责任。给其他方造成损失的，应赔偿其他方由此所遭受的直接经济损失。
3.乙方将尽力按照约定协助甲方成功交友。但基于婚恋服务行业的特点及乙方会员个人择偶意向的不可控性，本协议仅限甲方成功领证的返款依据。

2.若双方因履行本协议产生争议，应通过友好协商解决。若协商不成，双方同意任何一方有权向乙方所在地的人民法院提起诉讼解决。
